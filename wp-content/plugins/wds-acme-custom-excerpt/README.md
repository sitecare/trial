## Description ##

A plugin for creating and displaying custom excerpts.

## Installation ##

### Manual Installation ###

1. Upload the entire `/wds-acme-custom-excerpt` directory to the `/wp-content/plugins/` directory.
2. Activate WDS Acme Custom Excerpt through the 'Plugins' menu in WordPress.
